# Empty Proxy Rig

![Key Visual](assets/images/key-visual.png)

**Intuitive rig building addon using Empty objects**

A powerful toolkit for streamlining rigging workflows in Blender. Use Empty objects as proxies to intuitively position bones and set up IK constraints.

---

## Key Features

### 🎯 Interactive Dropper
Click on mesh surfaces to place Empties instantly. Use TAB to cycle through nearby Empties, Ctrl/Shift for vertex/edge snapping.

### 📍 Empty Placement
Place Empties at selected vertices, object origins, selection centers, and more.

### 🦴 Armature Tools
Automatically generate armatures from placed Empties. Use Adjust Mode to refine positions after creation.

### 🔗 Chain Order Manager
Visually manage bone chain order. Drag and drop to reorder, auto-sort by distance.

### ⚙️ IK Constraint Tools
Quickly add and configure IK constraints through a simple UI.

### 🎲 Volume Snap (v1.4.0+)
Advanced snapping for internal structures - snap to pipe centers, edge loop/ring centroids.

---

## Quick Start

1. **Install**: Blender → Edit → Preferences → Add-ons → Install
2. **Enable**: Search for "Empty Proxy Rig Suite" and check the box
3. **Access**: 3D Viewport → Sidebar (N) → "Empty Proxy Rig" tab
