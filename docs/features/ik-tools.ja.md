# IKツール

IK（Inverse Kinematics）コンストレイントを簡単に設定するためのツールです。

---

## パネルの表示条件

IK Constraint Tools パネルは、以下の条件で表示されます：

- アーマチュアを選択している
- **ポーズモード** になっている

---

## Quick Actions

### Add IK Constraint

選択したボーンにIKコンストレイントを追加します。

1. ポーズモードでボーンを選択
2. **Add IK Constraint** をクリック
3. ターゲット（Empty等）を設定

### Apply Visual Transform

IKによるポーズを現在の状態として確定します。

### Pose as Rest Pose

現在のポーズをレストポーズ（初期ポーズ）として設定します。

---

## IK設定項目

### Target

IKのターゲットオブジェクト。通常はEmptyを指定します。

### Chain Length

IKが影響するボーンの連鎖数。

- `0`: 自動（ルートボーンまで）
- `1以上`: 指定した数のボーンに影響

### Iterations

IK計算の反復回数。大きいほど精度が上がりますが、処理が重くなります。

### Use Stretch

有効にすると、ターゲットに届かない場合にボーンが伸びます。

### Use Rotation

有効にすると、ターゲットの回転もボーンに影響します。

### Influence

IKの影響度。0〜1の範囲で調整できます。

---

## Tips

!!! tip "Emptyをターゲットに設定"
    Target欄にEmptyをドラッグ＆ドロップできます。

!!! tip "複数ボーンを編集"
    複数のボーンを選択している場合、最後に選択したボーン（アクティブボーン）が編集対象になります。
    ++shift+"クリック"++ でアクティブボーンを変更できます。

---

## コンストレイントの削除

コンストレイントを削除するには：

1. パネルでコンストレイントの詳細を展開
2. 赤い **Delete Constraint** ボタンをクリック
