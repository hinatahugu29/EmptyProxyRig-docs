# Rig Mechanist へようこそ

**Rig Mechanist (リグ・メカニスト)** は、Blenderのリギングとモデリングワークフローに、CADのような「精度」と「構造的な堅牢さ」をもたらすために設計された統合アドオンスイートです。

以下の3つの専門ツールが連携して機能します：

1.  **EmptyProxyRig (v2)**: *[骨格]* - 構造設計と原点管理。
2.  **VisualSnapAssist (v4)**: *[精度]* - 幾何学的スナップと回転制御。
3.  **BoneIK_Tools**: *[動作]* - インスタントIKセットアップ。

---

## コンセプト

Blenderは直感的なモデリングに優れていますが、メカニカルリグやハードサーフェスモデリングで求められる「工学的」な操作感には課題があります。**Rig Mechanist** は以下の機能でその隙間を埋めます：

*   **幾何学的スナップ (Geometric Snapping)**: 仮想的な円の中心、接線、交点への正確なスナップ。
*   **プロキシリギング (Proxy Rigging)**: ボーンを作る前に「エンプティ」で設計を行い、完璧なピボット位置を確定させるワークフロー。
*   **視覚フィードバック (Visual Feedback)**: GPU描画によるオーバーレイで、スナップ位置や構造を明確に可視化。
