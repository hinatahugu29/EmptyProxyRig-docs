# Empty Proxy Rig

![キービジュアル](assets/images/key-visual.png)

**Emptyオブジェクトを使った直感的なリグ構築アドオン**

Blenderでのリギング作業を効率化する多機能ツールセットです。Emptyオブジェクトをプロキシとして活用し、直感的にボーンの位置決めやIKセットアップを行えます。

---

## 主な機能

### 🎯 Interactive Dropper
メッシュ表面をクリックするだけでEmptyを配置。TABキーで循環選択、Ctrl/Shiftで頂点・エッジモードに切り替え。

### 📍 Empty Placement
選択した頂点、原点、選択中心など、様々な位置にEmptyを配置。

### 🦴 Armature Tools
配置したEmptyから自動でアーマチュアを生成。Adjust Modeで後から調整も可能。

### 🔗 Chain Order Manager
ボーンチェーンの順序を視覚的に管理。ドラッグ＆ドロップで順序変更。

### ⚙️ IK Constraint Tools
IKコンストレイントの追加・設定をシンプルなUIで実行。

### 🎲 Volume Snap（v1.4.0+）
内部構造へのスナップ機能 - パイプ中心、エッジループ/リングの重心にスナップ。

---

## クイックスタート

1. **インストール**: Blender → Edit → Preferences → Add-ons → Install
2. **有効化**: 「Empty Proxy Rig Suite」を検索してチェック
3. **パネル表示**: 3D Viewport → サイドバー（N）→ 「Empty Proxy Rig」タブ

[詳しい使い方 →](getting-started.ja.md)
