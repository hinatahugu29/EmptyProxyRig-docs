# Empty Proxy Rig Suite v1.5.0 - 機能詳解 & 利用シーンガイド

このドキュメントは、各機能の役割と具体的な利用シーンをまとめたものです。

---

## 🎯 インタラクティブ配置 (Interactive Placement)

### 1. Start Empty Dropper
**機能の役割:**
直感的なクリック操作で、メッシュの表面や頂点、辺に合わせて連続してEmpty（アンカーポイント）を配置するツールです。

**利用シーン / こんな時に便利:**
- **メカの可動部設定:** ロボットの関節やピストンなど、正確な位置にリグの基準点を置きたいとき。
- **Volume Snapの活用:** パイプの中央や、円柱の中心など、通常のBlenderの機能では中心を取りにくい場所に一発で配置したいとき。
- **ラフなリギング:** 厳密な数値入力よりも、目で見て「ここ！」という場所に素早くボーンの元となる点を打ちたいとき。

### 2. Relocate Empty
**機能の役割:**
既に配置・設定済みのEmptyを、その属性（名前や親子関係など）を保ったまま別の場所に移動させる機能です。

**利用シーン / こんな時に便利:**
- **微調整:** 一通りリグを組んだ後で、「やっぱり関節位置をもう少しズラしたい」と思ったときに、リグを壊さずに位置だけ修正したいとき。
- **試行錯誤:** 複数のピボット位置案を試したいときに、セットアップを一からやり直すことなく位置だけ変更できるため、イテレーション速度が上がります。

## 📍 Empty配置 & 原点管理 (Placement & Origin)

### 3. Place at Selection / Origin
**機能の役割:**
選択している頂点やオブジェクトの原点に、即座にEmptyを作成します。

**利用シーン / こんな時に便利:**
- **正確な中心出し:** 選択したエッジループの中心に正確に配置したいとき（Selection Center）。
- **既存オブジェクトの活用:** 既に配置されているボルトやナットなどのパーツを基準点として利用したいとき。

### 4. Place & Move Origin / Set Origin to Empty
**機能の役割:**
Emptyの作成と同時に、対象となるメッシュの「原点（Origin）」もその位置に移動させます。

**利用シーン / こんな時に便利:**
- **ハードサーフェスリギングの準備:** メカパーツは回転軸が原点にあることが望ましいため、Emptyを配置すると同時にパーツの原点もそこに移動させることで、即座に回転アニメーションの準備が整います。
- **原点の修正:** モデリング中にズレてしまった原点を、意図したピボット位置（Emptyの位置）に再設定したいとき。

## 🦴 アーマチュア作成 (Armature Creation)

### 5. Create Armature from Empties
**機能の役割:**
配置した複数のEmptyを繋いで、一本の骨格（アーマチュア）を自動生成します。

**利用シーン / こんな時に便利:**
- **リグ生成の自動化:** 手動でボーンを一本ずつ押し出し（Extrude）して配置する手間を省略し、配置したマーカー（Empty）を結ぶだけでリグが完成します。
- **ピストンやシリンダー:** 始点と終点の2箇所にEmptyを置き、これを実行するだけで伸縮する構造の骨格が作れます。

### 6. Adjust Mode (自動リカバリー機能付)
**機能の役割:**
作成したアーマチュアのボーンと、元のEmptyを同期させます。Emptyを動かすとボーンも追従します。v1.5.0ではリグが壊れた際の自動復旧機能も強化されました。

**利用シーン / こんな時に便利:**
- **セットアップ後の修正:** アーマチュアを作った *後* で、「関節位置がズレていた」と気づいた場合でも、Emptyを動かすだけでリグ全体を修正できます。通常ならボーンのエディットモードに入って修正が必要な作業を、オブジェクトモードで直感的に行えます。

### 7. Reverse Adjust Mode (リバースモード)
**機能の役割:**
既存のアーマチュア（手動で作ったものなど）から、逆に配置用のEmptyを生成します。

**利用シーン / こんな時に便利:**
- **既存リグの改修:** 昔作ったリグや、他の人が作ったリグの構造を解析したり、このアドオンの機能を使って管理したい場合に、ボーン位置からEmptyを抽出して編集可能な状態にします。

## 🔗 チェーン順序管理 (Chain Order)

### 8. Auto Sort by Distance / Chain Management
**機能の役割:**
Empty同士の接続順序（親子関係の順序）を管理します。距離順での自動ソートや、手動での順序入れ替えが可能です。

**利用シーン / こんな時に便利:**
- **複雑な多関節アーム:** たくさんの関節があるロボットアームなどで、配置したEmptyの順番が意図した接続順になっていない場合、これを整理して正しいボーンの流れを作ります。
- **分岐構造:** どのEmptyが親で、どれが子になるかを視覚的に管理したいとき。

## 📦 管理 & 表示 (Management & Display)

### 9. Helper Collection Management
**機能の役割:**
作業用のEmptyを専用のコレクション("helper")にまとめ、一括で選択・非表示・削除を行います。

**利用シーン / こんな時に便利:**
- **シーンの整理整頓:** 作業中に増えすぎたEmptyが邪魔になったとき、ワンクリックで隠したり、リギング完了後に不要になったEmptyを一括削除してシーンをクリーンに保ちます。

### 10. Chain Preview (GPU可視化)
**機能の役割:**
Empty同士がどのどの順番で繋がる予定かを、3Dビューポート上に線と矢印で描画します。

**利用シーン / こんな時に便利:**
- **ミスの防止:** アーマチュアを作成する *前* に、「意図した通りの骨組みになるか」を視覚的に確認できます。おかしな接続（離れた場所に飛んでしまっているなど）を事前に発見できます。

## 🎛️ IKツール & その他 (IK & Tools)

### 11. Instant IK Setup
**機能の役割:**
選択したボーンに対して、ワンクリックでIK（インバースキネマティクス）を設定します。

**利用シーン / こんな時に便利:**
- **簡易リギング:** 本格的なリグを組むまでもないが、ちょっとした足や腕の動きをIKで制御したいとき、面倒なコンストレイント設定パネルを開くことなく即座にIK化できます。

### 12. Volume Snap (v1.4+)
**機能の役割:**
アドオン独自の強力なスナップ機能です。通常のBlenderのスナップでは不可能な「内部の中心」などを計算してスナップします。

**利用シーン / こんな時に便利:**
- **円筒・パイプの中心:** 頂点が存在しない「パイプの真ん中の空間」にピボットを置きたいとき。
- **厚みのある物体の中心:** 表面と裏面の中間点にボーンを通したいとき。ハードサーフェスモデリングにおいて非常に重宝します。
